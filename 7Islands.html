7 Islands Web Game

Description

Online Game developed with Spring Boot framework.

The game imitates the board game called 7 Islands, where you have to collect different treasures in order to win.

You can create an account in the web, and then create or join a game using a game-code.

The system will track different stadistic from your games, such as favorite island, or number of victories. In addition, you can get different achievements that will be shown in your profile.

You can also consult a ranking of best registered players in the web.
