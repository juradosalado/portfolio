Description

Artificial Inteligence bots for TicTacToe game and Knight and Knaves game.

In TicTacToe game, it will use a pygame window to run the game.

In Knight and Knaves game, you can define a Knight and Knaves puzzle and then the python script will print in console the result of those puzzles.
